# HACKberrySizeM_Mech

MサイズHACKberryの、機構部品の3Dプリント用STLファイルです。
ディレクトリ構成は以下のようになっています。

This repository contains stl files of middle sized HACKberry to make it with 3D printer.
The directory structure is below.

```
HbSizeSMech/
│  README.md        : この説明書です。/ This description file.
|  HbSizeM.stp      : MサイズHACKberryのCADファイルです。/ The design file of middle sized HACKberry
│
├─UnitHand/ : 手部分の部品のフォルダです。/ Directory for hand unit.
│  └─Unit_MR-H/ : 手のメインの部品類のフォルダです。/ Directory containing hand parts
│      ├─Parts_MR-H/ : 手のひら, 手の甲の部品のフォルダです。/ Directory containing palm parts
│      │      MR-H-01.stl
│      │      MR-H-02.stl
│      │      MR-H-03.stl
│      │      MR-H-04.stl

│      │      MR-H-12.stl
│      │      MR-H-13.stl
│      │      MR-H-14.stl
│      │      MR-H-15.stl
│      │
│      ├─Unit_MR-I/ : 人差し指の部品のフォルダです。/ Directory containing index finger parts
│      │      MR-I-01.stl
│      │      MR-I-02.stl
```
